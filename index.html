Métodos de Arrays em JavaScript

1. Array.prototype.shift()

Remove o primeiro elemento de um array e retorna esse elemento.

⚠️ Altera o array original!

const frutas = ['maçã', 'banana', 'laranja']; const primeiro = frutas.shift(); console.log(frutas); // ['banana', 'laranja'] console.log(primeiro); // 'maçã'

3. Array.prototype.concat()

Combina dois ou mais arrays e retorna um novo array com a combinação.

✅ Não altera os arrays originais!

const array1 = ['a', 'b']; const array2 = ['c', 'd']; const combinado = array1.concat(array2, ['e']); console.log(combinado); // ['a', 'b', 'c', 'd', 'e'] console.log(array1); // ['a', 'b'] (inalterado)

5. Array.prototype.flat()

Cria um novo array com todos elementos sub-arrays concatenados nele de forma recursiva até a profundidade especificada.

✅ Não altera o array original!

const array = [1, [2, [3, [4]]]]; const flat1 = array.flat(); // Profundidade padrão: 1 const flat2 = array.flat(2); // Profundidade: 2 const flatInf = array.flat(Infinity); // Todos os níveis console.log(flat1); // [1, 2, [3, [4]]] console.log(flat2); // [1, 2, 3, [4]] console.log(flatInf); // [1, 2, 3, 4]

6. Array.prototype.splice()

Altera um array adicionando/removendo elementos em posições específicas.

Retorna um array com os elementos removidos.

⚠️ Altera o array original!

const meses = ['Jan', 'Mar', 'Apr', 'Jun']; // Adiciona 'Feb' na posição 1, removendo 0 elementos meses.splice(1, 0, 'Feb'); // Remove 1 elemento na posição 4 const removidos = meses.splice(4, 1, 'May'); console.log(meses); // ['Jan', 'Feb', 'Mar', 'Apr', 'May'] console.log(removidos); // ['Jun']

8. Array.prototype.slice()

Retorna uma cópia superficial (shallow copy) de parte de um array.

✅ Não altera o array original!

const animais = ['ant', 'bison', 'camel', 'duck', 'elephant']; const parte = animais.slice(2, 4); const copia = animais.slice(); console.log(parte); // ['camel', 'duck'] console.log(copia); // ['ant', 'bison', 'camel', 'duck', 'elephant']
